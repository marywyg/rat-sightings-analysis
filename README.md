# Projekt analizy obserwacji szczurów (PL)

Projekt realizowany w ramach przedmiotu: Matematyka systemów cyfrowych i statystyka.

Projekt dzieli się na następujące etapy:

I. Analiza wstępna danych
1.	Utworzenie zbioru danych.
2.	Podstawowe statystyki danych.
3.	Analiza braków danych, przypadków ostających, ekstremalnych. Wpływ przypadków na statystyki.
4.	Wykresy ramka – wąsy.
II. Testy parametryczne i nieparametryczne
1.	Test istotności różnic (zmienne zależne i niezależne).
2.	Test zgodności chi-kwadrat (inne testy normalności rozkładu)
- histogram liczności, częstości.
3.	Przedział ufności dla średniej.
III. Współczynniki korelacji Pearsona i Spearmana
1.	Wyznaczenie i weryfikacja współczynników korelacji.
2.	Budowa modelu regresji liniowej i weryfikacja współczynników regresji liniowej.
IV. Analiza szeregów czasowych
1.	Trend szeregu czasowego (wygładzanie szeregu czasowego).
2.	Autokorelacja szeregu czasowego.
3.	Analiza szeregu czasowego – periodogram.
4.	Współczynniki sezonowości szeregu czasowego i dekompozycja szeregu czasowego.

# Rat sightings analysis project (ENG)

Project carried out as part of the subject: Mathematics of digital systems and statistics.

The project is divided into the following stages:

I. Preliminary Data Analysis
1. Creation of the dataset.
2. Basic data statistics.
3. Analysis of missing data, residual cases, and extreme cases. The influence of cases on statistics.
4. Box-and-whisker plots.
II. Parametric and nonparametric tests
1. Test for significance of differences (dependent and independent variables).
2. Chi-square goodness-of-fit test (other tests of normality)
- histogram of frequencies.
3. Confidence interval for the mean.
III. Pearson and Spearman correlation coefficients
1. Determination and verification of correlation coefficients.
2. Construction of a linear regression model and verification of linear regression coefficients.
IV. Time series analysis
1. Time series trend (time series smoothing).
2. Time series autocorrelation. 3. Time series analysis – periodogram. 4. Time series seasonality coefficients and time series decomposition.
